应为 DescribeRegionsRequest.DescribeRegionsRequest

2、就是看到 DescribeLoadBalancerAttribute api 返回的json数据中存在内容一样的两个属性：ListenerPortsAndProtocal 和 ListenerPortsAndProtocol
